# 2.2 Cannot Send Email

## Issue Summary

Customer unable to send email messages.

## Response Template

```
Hello [Customer Name],

I understand you're having trouble sending emails. Let's resolve this together.

Please check the following:

1. 

Common error codes:
- 550: Recipient rejected
- 553: Invalid recipient address
- 554: Message rejected (often spam filter)

If the issue persists, please send me:
- The exact error message
- Screenshot if possible
- Recipient email address

Best regards,
[Your Name]
Support Team
```

## Troubleshooting Steps

1. **Check credentials** - Username/password
2. **Verify SMTP settings** - Server, port, SSL/TLS
3. **Check quotas** - Storage limits
4. **Review error codes** - 5xx codes
5. **Authentication** - SPF/DKIM issues

## Common Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| 550 | Mailbox unavailable | Check recipient address |
| 553 | Invalid address | Verify email format |
| 554 | Rejected | Check spam filters |
