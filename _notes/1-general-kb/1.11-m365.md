# 1.11 Microsoft 365

## Overview

Cloud-based email and collaboration platform.

## Key Features

- Exchange Online
- Email security
- Compliance
- Anti-spam/malware

## Admin Tasks

### Mail Flow Rules

Admin Center → Exchange → Mail flow → Rules

### Connectors

For hybrid or custom routing:
- Inbound connector
- Outbound connector

### Message Trace

Track email delivery:
1. Exchange Admin Center
2. Mail flow → Message trace
3. Search by sender/recipient/date

## Common Commands

```powershell
# Connect to Exchange Online
Connect-ExchangeOnline

# Get mailbox
Get-Mailbox user@example.com

# Message trace
Get-MessageTrace -SenderAddress sender@example.com
```
